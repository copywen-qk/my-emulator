# RV32IM CPU 驗證測試報告

## 測試環境
- 日期：2026-04-10
- 專案：nemu-rv32
- CPU：RV32IM 單週期實現
- 工具：Verilator 5.020

## 實現功能

### ✅ 已完成階段
1. **指令提取 (Fetch)**
   - PC 管理
   - 記憶體讀取介面
   - 指令緩衝

2. **指令解碼 (Decode)**
   - RISC-V 指令欄位解碼
   - 暫存器檔案讀取
   - 立即數生成
   - 操作數選擇

3. **執行階段 (Execute)**
   - ALU 運算：
     - 算術：ADD, SUB, ADDI
     - 邏輯：AND, OR, XOR, ANDI, ORI, XORI
     - 移位：SLL, SRL, SRA, SLLI, SRLI, SRAI
     - 比較：SLT, SLTU, SLTI, SLTIU
   - 分支判斷

4. **記憶體存取 (Memory)**
   - Load 指令：LB, LH, LW, LBU, LHU
   - Store 指令：SB, SH, SW
   - 記憶體對齊處理

5. **寫回階段 (Writeback)**
   - 暫存器寫入控制
   - 結果選擇（ALU 結果 vs 記憶體讀取）

6. **控制流**
   - PC 更新邏輯
   - 分支預測（簡單）
   - JAL/JALR 處理

### ⚠️ 待實現功能
1. **M 擴展**（乘除法）
2. **A 擴展**（原子操作）
3. **CSR 暫存器**
4. **中斷與例外處理**
5. **流水線優化**
